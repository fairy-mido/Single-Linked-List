This Project is meant to get you understand the idea of how to implement Single Linked List in C Language.
Also, I added some conditions to make it understandable to read. I hope you find it useful :>
There is an app.exe, you can run the app from cmd or any IDE as you wish.

More Information About How to Access CMD & Compile if you want to change in the code itself:
-> gcc main.c single_LL.c single_LL.h -o app  
This Command is compiling your code and give an exe named by app
-> app
Open The app Cmd Page to see your Code.
